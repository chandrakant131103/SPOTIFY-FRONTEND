# 🎧 Spotify Clone - Frontend (React)

A high-performance, responsive UI built with React, featuring a frosted-glass aesthetic and seamless music streaming.

## 🎨 Design Features
- **Glassmorphism UI**: High-blur background with semi-transparent panels.
- **Responsive Layout**: Mobile-first design with bottom navigation.
- **Real-time Auth**: Token-based persistence using LocalStorage.

## 🚀 Tech Stack
- **React (Vite)**: Frontend framework.
- **Axios**: API communication with interceptors for JWT.
- **Lucide React**: For sleek, modern icons.
- **CSS3 Variables**: For consistent theme management.

## 🔧 Installation
1. `npm install`
2. Create `src/api/axiosConfig.js` and point to your Render URL.
3. `npm run dev`

## 📱 Mobile Support
The UI automatically adapts to mobile screens (max-width: 768px), transforming the sidebar into a bottom navigation bar for a native app experience.
